Question 2easy

அகப்பொருள் இலக்கியம் என்பது எதை பற்றியது?

  1. போர்
  2. காதல்
  3. கடவுள்
  4. இயற்கை
Show answer

Answer: காதல்

அகப்பொருள் காதல் (காதல் வாழ்க்கை) பற்றியது. புறப்பொருள் போர், வீரம், கொடை பற்றியது. சங்க இலக்கியம் அக, புறம் என இரு பிரிவுகளில் அமைந்துள்ளது.

Question 6medium

  2. அன்பு, காதல் பற்றிய கவிதைகள்

  4. கடவுள் பற்றிய கவிதைகள்
Show answer

Answer: அன்பு, காதல் பற்றிய கவிதைகள்

சங்க இலக்கியத்தில் 'அகம்' என்பது காதல், அன்பு, மனத்தின் உணர்வுகளை பற்றிய கவிதைகளைக் குறிக்கும். 'புறம்' என்பது வீரம், போர், கொடை, சமூகம் பற்றியவையாகும்.

Question 7medium

  1. ஐந்து வகை போர்முறைகள்
  2. காதல் வாழ்க்கையின் ஐந்து நிலைகளை வெவ்வேறு நிலங்களுடன் தொடர்புபடுத்திய ஐந்து வகை: குறிஞ்சி, முல்லை, மருதம், நெய்தல், பாலை
  3. ஐந்து வகை கவிதை வடிவங்கள்
  4. ஐந்து சங்கப் புலவர்கள்
Show answer

Answer: காதல் வாழ்க்கையின் ஐந்து நிலைகளை வெவ்வேறு நிலங்களுடன் தொடர்புபடுத்திய ஐந்து வகை: குறிஞ்சி, முல்லை, மருதம், நெய்தல், பாலை

Question 8medium

'புறநானூறு' என்ற நூல் எதைப் பற்றியது?

  1. காதல் கவிதைகள்
  2. வீரம், கொடை, போர், அரசர்களைப் பற்றிய நானூறு (400) புறத்திணைப் பாடல்கள்
  3. இறைவனைப் பற்றிய பாடல்கள்
  4. நீதி கவிதைகள்
Show answer

Answer: வீரம், கொடை, போர், அரசர்களைப் பற்றிய நானூறு (400) புறத்திணைப் பாடல்கள்

புறநானூறு — 400 பாடல்கள் கொண்ட சங்க நூல். வீரம் (தும்பை, வாகை), கொடை (காஞ்சி), போர், அரசர்களின் புகழ், மரணம் ஆகியவற்றை பற்றிய 'புறத்திணை' கவிதைகள். பல புலவர்களால் இயற்றப்பட்டவை.

  3. நூறு புலவர்கள் இயற்றிய நூல்

Question 12medium
