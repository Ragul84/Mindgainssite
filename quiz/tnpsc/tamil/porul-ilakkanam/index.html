Question 1medium

அகத்திணைகள் எத்தனை?

  1. 5
  2. 7
  3. 8
  4. 10
Show answer

Answer: 5

அகத்திணைகள் 5: குறிஞ்சி (மலை - காதல் ஒருமித்தல்), முல்லை (காடு - இரவு கூட்டல்), மருதம் (வயல் - ஊடல்), நெய்தல் (கடல் - இரங்கல்), பாலை (வழி - பிரிதல்).

Question 2medium

குறிஞ்சித் திணை எந்த நில/காலத்தை குறிக்கிறது?

  1. கடல் - பகல்
  2. மலை - கூதிர்/முன்பனி
  3. வயல் - முன்பனி
  4. காடு - கார்காலம்
Show answer

Answer: மலை - கூதிர்/முன்பனி

குறிஞ்சித் திணை: நிலம் - மலையும் மலை சார்ந்த இடமும், காலம் - கூதிர் மற்றும் முன்பனிக்காலம், பூ - குறிஞ்சி, உரிப்பொருள் - புணர்தல் (காதல் ஒருமித்தல்).

  2. புறப்பொருள்
  3. பக்தி இலக்கியம்
  4. நீதி இலக்கியம்

Question 4easy

'ஆடு மேய்த்தவன் ஆனை மேய்க்கலாம்' — இந்த பழமொழியின் கருத்து என்ன?

  1. சிறிய வேலையிலிருந்து பெரிய வேலைக்கு செல்லலாம்
  2. ஆடும் யானையும் சேர்ந்து மேயும்
  3. கால்நடை வளர்ப்பு முக்கியம்
  4. உழைப்பால் எதுவும் செய்யலாம்
Show answer

Answer: சிறிய வேலையிலிருந்து பெரிய வேலைக்கு செல்லலாம்

'ஆடு மேய்த்தவன் ஆனை மேய்க்கலாம்' — சிறிய அனுபவம் பெற்றவர்கள் பெரிய பொறுப்புகளையும் ஏற்கலாம் என்பது இப்பழமொழியின் பொருள். திறமை வளரும் என்ற நம்பிக்கையை வெளிப்படுத்துகிறது.

Question 5easy

  1. யானை ஊர்வலம்
  2. பெரியோர் வரும் முன் அவர்களின் புகழ் முன்னரே வரும்
  3. ஒலியின் வேகம் யானையை விட வேகமானது

Show answer

Answer: பெரியோர் வரும் முன் அவர்களின் புகழ் முன்னரே வரும்

'யானை வரும் பின்னே மணி ஓசை வரும் முன்னே' — சிறந்தவர்கள் / பெரியவர்கள் வருவதற்கு முன்பே அவர்களின் புகழ் / செய்தி முன்னரே எட்டும் என்பது இதன் பொருள்.

‘ஓர் எழுத்து பல பொருட்படுதல்’ எனும் இலக்கணக் கோட்பாட்டை விளக்கும் நூல் எது?

  1. நன்னூல்
  2. தொல்காப்பியம்
  3. வீரசோழியம்
  4. கலித்தொகை
Show answer

Answer: தொல்காப்பியம்

தொல்காப்பியம் எழுத்ததிகாரத்தில் ‘ஓரெழுத்து பல பொருட்படுதல்’ பற்றி விளக்குகிறது.

Question 77hard

  1. ஏழு
  2. எட்டு
  3. ஒன்பது
  4. பத்து
Show answer

Answer: எட்டு

தொன்னூல் விளக்கம் புறப்பொருளை எட்டு வகையாகப் பகுத்துக் காட்டுகிறது (வெட்சி முதல் குடை வரை).

Question 78hard

‘செவிலி தோழிக்கு உரைத்தல்’ என்னும் துறை குறித்தொழுக்கத்தின் எந்தப் பகுதியில் அமைந்துள்ளது?

  1. பரத்தையிற் பிரிவு
  2. ஊடல் நீடல்
  3. காதலர் பிரிவு
  4. இரவுக்குறி
Show answer

Answer: காதலர் பிரிவு

காதலர் பிரிவில் தலைவி வருந்தும்போது செவிலி தோழிக்கு உரைப்பதாக இத்துறை அமையும்.

  2. அதிகாரம் 85

திருக்குறள் அதிகாரம் 90 ‘இன்னாசெய்யாமை’ எனப் பெயரிடப்பட்டுள்ளது.
