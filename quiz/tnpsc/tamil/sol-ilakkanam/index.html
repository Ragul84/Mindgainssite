Question 1medium

'கை + அமர்த்தல்' என்பது புணர்ந்து என்னவாகும்?

  1. கையமர்த்தல்
  2. கையமர்தல்
  3. கைஐமர்த்தல்
  4. கைஐ அமர்த்தல்
Show answer

Answer: கையமர்த்தல்

கை + அமர்த்தல் = கையமர்த்தல். இங்கு 'கை' என்ற சொல்லில் உள்ள 'ய்' என்ற மெய் 'அ' என்ற உயிர் சேர்வதால் 'ய' என்று திரியும் — இது உயிர் புணர்ச்சி.

Question 2medium

'பெயர்ச்சொல்' என்பது தமிழ் இலக்கணத்தில் என்ன?

  1. வினையை குறிக்கும் சொல்
  2. பெயரைக் குறிக்கும் சொல் — உயிரினம் அல்லது உயிரற்றதின் பெயர்
  3. இடத்தை மட்டும் குறிக்கும் சொல்
  4. காலத்தை குறிக்கும் சொல்
Show answer

Answer: பெயரைக் குறிக்கும் சொல் — உயிரினம் அல்லது உயிரற்றதின் பெயர்

பெயர்ச்சொல் — பெயரை (நபர், இடம், பொருள், உயிர் அல்லது உயிரற்றது) குறிக்கும் சொல். தமிழில் பெயர்ச்சொல் ஆறு வகை: பொருட்பெயர், இடப்பெயர், காலப்பெயர், சினைப்பெயர், குணப்பெயர், தொழிற்பெயர்.

Question 3medium

'வினைச்சொல்' என்பது என்ன?

  1. பெயரைக் குறிக்கும் சொல்
  2. செயலை அல்லது நிலையை குறிக்கும் சொல்
  3. இடத்தைக் குறிக்கும் சொல்
  4. அளவைக் குறிக்கும் சொல்
Show answer

Answer: செயலை அல்லது நிலையை குறிக்கும் சொல்

வினைச்சொல் — ஒரு செயல் அல்லது நிலையை குறிக்கும் சொல். தமிழில் வினை இரண்டு வகை: செய்வினை (வேண்டுகோள் விடுத்தல், செயல் செய்தல்), செயப்பாட்டு வினை. காலம் மூன்று: இறந்தகாலம், நிகழ்காலம், எதிர்காலம்.

Question 4medium

  1. பெயர்ச்சொல்லுக்கும் வினைச்சொல்லுக்கும் இடையே வரும் சொல் — ஒட்டுகள், விகுதிகள்
  2. மக்களை குறிக்கும் சொல்
  3. இடத்தை குறிக்கும் சொல் மட்டுமே
  4. கவிதையில் வரும் சொல்
Show answer

Answer: பெயர்ச்சொல்லுக்கும் வினைச்சொல்லுக்கும் இடையே வரும் சொல் — ஒட்டுகள், விகுதிகள்

இடைச்சொல் — தமிழில் பெயர்ச்சொல் அல்லது வினைச்சொல்லுக்கு முன்னோ பின்னோ சேர்க்கப்படும் சொல். உதாரணம்: 'ஆல்', 'உம்', 'ஓ', 'கொல்' போன்றவை. இவை தனியாக பொருள் தராது.

Question 6medium

  2. கி.மு. 3ஆம் நூற்றாண்டு (சுமார்)
  3. கி.பி. 12ஆம் நூற்றாண்டு
  4. கி.மு. 1000ஆம் ஆண்டு
Show answer

Answer: கி.மு. 3ஆம் நூற்றாண்டு (சுமார்)

தொல்காப்பியரின் காலம் சுமார் கி.மு. 3ஆம் நூற்றாண்டு என்று பரவலாக ஏற்றுக்கொள்ளப்படுகிறது (சிலர் கி.மு. 5 – கி.மு. 3 என்பர்). இவர் 'அகத்தியர்' சீடர் என்று சொல்வர்.

  2. கி.பி. 13ஆம் நூற்றாண்டில் பவணந்தி முனிவர் இயற்றிய தமிழ் இலக்கண நூல்
  3. அவ்வையார் நூல்

Question 8medium

தமிழ் மொழியில் 'வேர்ச்சொல்' என்றால் என்ன?

  1. மரத்தின் வேரை குறிக்கும் சொல்
  2. சொல்லின் அடிப்படை வடிவம் — பிரதா, பிரதிமை நீக்கிய சொல்
  3. வீரத்தை குறிக்கும் சொல்
  4. வேதத்தில் இருந்து வந்த சொல்
Show answer

Answer: சொல்லின் அடிப்படை வடிவம் — பிரதா, பிரதிமை நீக்கிய சொல்

வேர்ச்சொல் (Root word) — ஒரு சொல்லின் அடிப்படை வடிவம். இதிலிருந்து பல்வேறு சொற்கள் உருவாகும். உதாரணம்: 'செய்' என்பது வேர்ச்சொல்; செய்தான், செய்கிறான், செய்வான் என்று மாறும்.

Question 9hard

  2. சொற்களின் செயல்பாட்டை (subject, object, etc.) குறிக்க பயன்படும் சுட்டுகள் — ஐ, ஆல், கு, இன், இல், உடன், அது
  3. மொழி வேறுபாடுகள்
  4. கவிதை வகைகள்
Show answer

Answer: சொற்களின் செயல்பாட்டை (subject, object, etc.) குறிக்க பயன்படும் சுட்டுகள் — ஐ, ஆல், கு, இன், இல், உடன், அது

தமிழில் வேற்றுமை என்பது சொல்லுக்குப் பின் வரும் உருபுகளால் (vibhakti) சொல்லின் பங்கு தெரிகிறது. 8 வேற்றுமைகள்: எழுவாய் (முதன்மை), ஐ (object), ஆல் (by), கு (to/for), இன் (from/of), இல் (in), உடன் (with), ஏ (vocative).

Question 10hard

தமிழில் 'வினையெச்சம்' என்பது என்ன?

  2. ஒரு வினை நடந்து மற்றொரு வினை நடைபெறும் போது முதல் வினையை குறிக்கும் வடிவம்
  3. கேள்விக்கு பதில் சொல்லும் வினை

Answer: ஒரு வினை நடந்து மற்றொரு வினை நடைபெறும் போது முதல் வினையை குறிக்கும் வடிவம்
