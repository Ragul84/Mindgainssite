Question 13easy

  2. The mother only
  3. Both equally
  4. Temperature during pregnancy
Show answer

Answer: The father's sperm — X-bearing sperm gives girl (XX), Y-bearing sperm gives boy (XY)

Human sex determination: females are XX, males are XY. All eggs carry X chromosome; sperm carry either X or Y. If Y-bearing sperm fertilizes egg → XY (boy). If X-bearing sperm → XX (girl). So the father's sperm determines the child's sex, not the mother.

Question 14easy

Binomial nomenclature (scientific naming of organisms) uses:

  1. Genus name (capitalized) + species name (lowercase) in Latin/Greek, italicized — e.g., Homo sapiens, Panthera tigris
  2. Common name only
  3. Only one Latin name
  4. Kingdom and phylum names
Show answer

Answer: Genus name (capitalized) + species name (lowercase) in Latin/Greek, italicized — e.g., Homo sapiens, Panthera tigris

Carl Linnaeus (1758) established binomial nomenclature: each organism gets a two-part Latin name — genus (capitalized) + specific epithet (lowercase). Example: Homo sapiens (humans), Panthera tigris (tiger), Oryza sativa (rice), Mangifera indica (mango). Universally accepted system.

  3. Heart fails

Show answer

Answer: Kidneys fail to filter waste from blood (renal failure) — dialysis artificially filters blood using a semipermeable membrane
