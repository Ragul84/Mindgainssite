Question 1easy

In a code, BOOK is written as CPPL. How is DOOR written?

  1. EPPS
  2. EPPQ
  3. EQPS
  4. EQPR
Show answer

Answer: EPPS

Each letter is shifted by +1: B→C, O→P, O→P, K→L. So D→E, O→P, O→P, R→S. DOOR = EPPS.

Question 8medium

  1. 36
  2. 40

  4. 60
