Some erasers are rulers. 4. All rulers are scales. Which of the following conclusions logically follow from the given statements?

  1. Only 1 and 2 follow
  2. Only 1, 2 and 3 follow
  3. Only 1, 2 and 4 follow
  4. All 1, 2, 3 and 4 follow
Show answer

Answer: Only 1, 2 and 3 follow

From I and II, Some pens are erasers. From II and III, No pencil is a scale. From III and IV, No eraser is a scale, but some rulers are scales, so Some erasers are not rulers is definite, but Some erasers are rulers is possible (hence not definite). All rulers are scales is a restatement of IV, which is not a conclusion but a given statement itself. Therefore, only 1 and 2 definitely follow. For 3, it's possible but not definite given the 'No eraser is a scale' and 'All scales are rulers' (some rulers could be erasers indirectly). Re-evaluating 3: Since 'All scales are rulers' and 'No eraser is a scale', it means no eraser is a scale, so if an eraser is a ruler, that ruler cannot be a scale. This is possible. Hence, 3 is not a definite conclusion. Let's re-examine 3. From 'No eraser is a scale' and 'All scales are rulers', it implies that rulers which are scales are not erasers. However, there can be rulers which are not scales, and those rulers could be erasers. So, some rulers are erasers is possible, but 'Some erasers are rulers' is not a definite conclusion. Let's reconsider. From No eraser is a scale (III) and All scales are rulers (IV), it means that any scale is a ruler, and no eraser is a scale. This means that rulers which are scales cannot be erasers. However, it does not preclude some rulers (which are not scales) from being erasers. So, 'Some erasers are rulers' is not a definite conclusion. Therefore, only 1 and 2 follow.
