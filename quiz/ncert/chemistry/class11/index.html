  2. pH = 7.00
  3. pH = 9.24
  4. pH = 11.24
Show answer

Answer: pH = 9.24

The equilibrium is NH₃ + H₂O ⇌ NH₄⁺ + OH⁻. Kb = [NH₄⁺][OH⁻] / [NH₃].   2. pH = pKa

  4. pH = 14 - pKa

Answer: pH = pKa

The Henderson-Hasselbalch equation for a basic buffer is pOH = pKb + log([Conjugate Acid]/[Base]). Alternatively, for an acidic buffer (using the conjugate acid's pKa), pH = pKa + log([Base]/[Conjugate Acid]). If [Base] = [Conjugate Acid], then the ratio [Base]/[Conjugate Acid] = 1. Since log(1) = 0, the equation simplifies to pH = pKa. Thus, the pH of the buffer solution will be equal to the pKa of the conjugate acid.
